# 扫描宝 · Scan Treasure —— 开源文档/证件扫描工具（H5）

> 一个**免费、开源、纯本地**的文档与证件扫描 Web 应用。
> 拍照后自动摆正、增强、导出 PDF，**所有处理在浏览器本地完成，图片永远不上传**。

## ✨ 特性
- 🔒 **100% 本地处理**：图片从不离开你的设备，无服务器、无上传
- 🆓 **完全免费**：无广告、无登录、无内购
- 📱 **扫码即用**：手机浏览器打开即用，无需安装 App
- 🧑‍💻 **开源（MIT）**：可自行部署、二次开发

## 🛠 技术栈
Vue 3 + Vite + TypeScript + Vant 4 + opencv.js + jsPDF（+ Tesseract.js 可选 OCR）
部署：GitHub Pages（静态托管）
