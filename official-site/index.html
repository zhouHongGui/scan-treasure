开源项目

适合个人使用，也适合自行部署

扫描宝前端应用使用 Vue 3、Vite、TypeScript、Vant、opencv.js 和 jsPDF 构建。当前官网目录则使用纯 HTML、CSS、JavaScript，便于百度等搜索引擎直接抓取首页正文。

SEO 结构

官网首页为静态 HTML，正文、标题、FAQ、结构化数据和 robots.txt 均不依赖前端渲染。上线后填写真实域名、提交百度资源平台，并补充 sitemap。

常见问题

关于扫描宝的几个答案

扫描宝会上传我的证件或合同照片吗？

不会。项目定位是浏览器本地扫描工具，图片读取、边缘识别、校正、增强和 PDF 导出都在本地完成。

扫描宝是免费的吗？

是。扫描宝是免费开源项目，没有登录、广告和内购。你也可以基于 MIT 协议自行部署或二次开发。

它适合做哪些文档扫描？

适合整理自己的合同、票据、课堂笔记、书页、身份证照片和其他合法持有的纸质材料。

为什么官网不用 Vue 渲染？

百度 SEO 更看重首屏 HTML 可抓取内容。官网使用静态 HTML 输出完整正文，Vue 应用只负责实际扫描工具，两者分开更稳。
