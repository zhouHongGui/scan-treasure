开源项目

适合个人使用，也适合自行部署

扫描宝前端应用使用 Vue 3、Vite、TypeScript、Vant、opencv.js 和 jsPDF 构建。当前官网目录使用纯 HTML、CSS、JavaScript，结构简单，便于维护和静态部署。

部署方式

官网可直接作为静态文件托管，也可以和扫描工具分开部署。上线前只需要替换真实访问地址、图标和项目说明。

常见问题

关于扫描宝的几个答案

扫描宝会上传我的证件或合同照片吗？

不会。项目定位是浏览器本地扫描工具，图片读取、边缘识别、校正、增强和 PDF 导出都在本地完成。

扫描宝是免费的吗？

是。扫描宝核心工具免费开源，无需登录，工具内无内购。官网底部可能展示清晰标注的外部推广入口，你也可以基于 MIT 协议自行部署或二次开发。

它适合做哪些文档扫描？

适合整理自己的合同、票据、课堂笔记、书页、身份证照片和其他合法持有的纸质材料。

为什么官网不用 Vue 渲染？

官网内容较轻，使用静态 HTML 可以减少构建复杂度。Vue 应用专注实际扫描工具，两者分开维护更清晰。
